Rust WASM Host Boundary Bug PoC

This PoC demonstrates a browser-side bug where the host passes a pointer and a length to WASM and the module trusts the length.

The backend is simulated in Rust/WASM. The request payload is stored in a fixed-size 16-byte buffer. If the host report is too large, it can overflow the adjacent authentication flag and bypass access control.
