Browser WASM Login Demo

This demo simulates a real login page where the browser sends credentials to a backend.

The server-side login logic verifies the username and password, then sets authentication state. In this demo, the backend is simulated in WASM.

The submitted password is copied into a fixed-size 16-byte buffer. A too-long password can overflow into the adjacent `isAdmin` flag and bypass authentication.
